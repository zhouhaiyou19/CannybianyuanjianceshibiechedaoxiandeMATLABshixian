# Canny边缘检测识别车道线的MATLAB实现

本资源文件提供了一个基于MATLAB的Canny边缘检测算法实现，用于识别图像中的车道线。通过该实现，您可以获得以下四张图像：

1. **原始图像的梯度幅值图**：使用Sobel算子计算原始图像的梯度幅值图。
2. **非极大值抑制后的梯度幅值图**：对梯度幅值图进行非极大值抑制，以细化边缘。
3. **双阈值处理和搜索连接后的梯度幅值图**：应用双阈值处理和搜索连接算法，进一步筛选和连接边缘。
4. **最后的梯度二值化图**：通过选取合适的阈值，生成最终的二值化图像，尽可能保留车道线并减少其他非车道线的干扰。

该实现适用于处理名为“lanes”的图像，帮助您更好地理解和应用Canny边缘检测算法进行车道线识别。
